This app takes you through a flow of screens, allowing you to login, or create a username and password. From either page you are then directed to a dashboard page, in which your current username and password are displayed.

Files:
app.py - Flask app to be run via mod_wsgi
config.py — config file
lost_conifg.json — json config file

templates/
	login.html
	create_user.html
	dashboard.html
